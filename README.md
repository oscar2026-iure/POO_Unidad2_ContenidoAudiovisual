# POO_Unidad2_ContenidoAudiovisual
## 1. Descripción del Proyecto
Este proyecto es una aplicación de consola desarrollada en Java bajo el paradigma de **Programación Orientada a Objetos**. Su propósito es gestionar contenidos audiovisuales (Películas, Series, Documentales y Ciberseguridad) aplicando herencia, polimorfismo y relaciones de agregación/asociación
### Problema que resuelve:
La aplicación resuelve la necesidad de centralizar información técnica de diferentes formatos de video, permitiendo la extensibilidad del sistema. En esta versión, se ha integrado contenido de **Ciberseguridad** para ayudar a organizar recursos educativos sobre respuesta a incidentes (CSIRT) y aprendizaje técnico.
### Características Principales:
* **Herencia y Polimorfismo**: Uso de clases abstractas para una base sólida de datos.
* **Relaciones Avanzadas**: Implementación de Asociación y Agregación.
* **Contenido Especializado**: Clases propias enfocadas en seguridad informática.
